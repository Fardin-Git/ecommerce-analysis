# ecommerce-analysis

## Procedure
  1. Clean, reshape and expand the sales data set according to our wishes
  2. Conduct a thorough analysis of the cleaned dataset, gain insights, and derive a suitable recommendation for action.
  3. Perform SQL aggregations on the sales dataset and save the aggregated datasets that are relevant for our recommendation in new CSV files, so they             are ready for Power BI.
  4. Lastly, all relevant data sets are appropriately visualized in Power BI.

## Folder Structure
- data: Raw data, cleaned data and Power BI data
- notebooks: Python-Notebooks
- database: SQLite database
- powerbi: Power BI Dashboard
- sql: SQL-queries

## Tools
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQLite
- Power BI
